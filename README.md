# Silbenteppich - Lernspiel

Ein browserbasiertes Lernspiel zum Silbenlesen fuer Kinder mit Lese-/Rechtschreibschwaeche (LRS). Das Spiel liest Silben und Woerter vor, das Kind klickt die richtige Antwort. Das System passt sich automatisch an den Lernfortschritt an.

**[Jetzt spielen](https://v3nomsoup.github.io/silbenlernspiel/)**

## Features

- Silbenteppich mit bunten, klickbaren Kacheln
- Deutsche Sprachausgabe (eigene Aufnahmen oder Web Speech API)
- Adaptives Schwierigkeitssystem (6 Level: offene Silben bis dreisilbige Woerter)
- Punkte mit Geschwindigkeitsbonus und Streak-Multiplikator
- Puzzle-Belohnungssystem mit kindgerechten SVG-Bildern
- Spielstand wird im Browser gespeichert
- Funktioniert auf Desktop, Tablet und Handy
- Kein Server noetig — laeuft komplett im Browser

## Aufnahme-Tool

Unter `recorder.html` gibt es ein Tool zum Einsprechen eigener Silben und Woerter. Exportierte Aufnahmen koennen als `audio/recordings.json` committed werden und werden dann automatisch im Spiel verwendet.

## Tech Stack

Vanilla HTML, CSS und JavaScript — kein Build-Tool, keine Abhaengigkeiten.
